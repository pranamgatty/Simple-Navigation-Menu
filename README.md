# Simple-Navigation-Menu
A modern, interactive navigation component with scroll-triggered styling and hover animations. Built with vanilla HTML, CSS, and JavaScript—no dependencies.

## Project Structure<br>
├── index.html    # Semantic HTML structure <br>
├── styles.css    # CSS3 with custom properties and animations<br>
└── script.js     # Vanilla JavaScript for interactivity

## How It Works<br>
-Scroll event listener detects when user scrolls past 30px threshold.<br>
-Header applies .scrolled class to trigger background color change and shadow effect.<br>
-Menu links trigger aqua-colored shadow on hover with smooth transitions.<br>
-Box shadow adjusts based on scroll state for visual consistency.
